# 한국어 교정·교열 룰북

객관적 오류만 고치고 뜻·어투·격식·문단 구조는 유지한다. **강**은 정답이 거의 하나인 오류, **문맥**은 의미·허용 표기·전문 분야를 확인해야 하는 항목이다. 규범으로 설명할 수 없는 어색함은 문장 윤문 단계로 넘긴다.

## G-1. `예요/이에요/에요` [강]

받침이 없으면 `-예요`, 받침이 있으면 `-이에요`를 쓴다. `-에요`는 `아니에요`에 쓴다.

- `도구에요` → `도구예요`
- `책이예요` → `책이에요`
- `아니예요` → `아니에요`

## G-2. `되/돼`, `뵈/봬` [문맥]

`되어`로 풀리면 `돼`, 어간이면 `되`다. `뵈어요`의 준말은 `봬요`다.

- `잘 됬다` → `잘됐다`
- `그러면 안 되` → `그러면 안 돼`
- `빨리 돼고 싶다` → `빨리 되고 싶다`
- `내일 뵈요` → `내일 봬요`

## G-3. `안/않` [문맥]

부사 `안`은 띄고, 용언 어간 `않-`은 앞말과 결합한다.

- `밥을 않 먹었다` → `밥을 안 먹었다`
- `가지 안았다` → `가지 않았다`

## G-4. `-데/-대` [문맥]

직접 겪은 일을 회상하면 `-데`, 남의 말을 전하면 `-대`다. 문맥 없이 형태만 보고 고치지 않는다.

- 직접 보고 감탄: `사람이 많데`
- 전언: `사람이 많대`

## G-5. `-든지/-던` [문맥]

선택·무관은 `-든지`, 과거 회상은 `-던`이다.

- `가던지 말던지` → `가든지 말든지`
- `어릴 때 놀든 곳` → `어릴 때 놀던 곳`

## G-6. `-로서/-로써` [문맥]

자격·지위는 `-로서`, 도구·수단은 `-로써`다.

- `학생으로써 본분` → `학생으로서 본분`
- `대화로서 해결` → `대화로써 해결`

## G-7. `율/률`, `열/렬` [강]

모음이나 `ㄴ` 받침 뒤에는 `율·열`, 그 밖의 받침 뒤에는 `률·렬`을 쓴다.

- `비률` → `비율`, `확율` → `확률`
- `나렬` → `나열`; 줄 세우기 뜻의 `정열` → `정렬`

열정이라는 뜻의 명사 `정열`은 올바른 별개 단어이므로 고치지 않는다.

## G-8. `채/체/째` [문맥]

그 상태로는 `채`, 척함은 `체`, 차례·전부는 `째`다.

- `신발을 신은 체` → `신발을 신은 채`
- `못 본 채하다` → `못 본 체하다`
- `껍질채` → `껍질째`

## G-9. `맞히다/맞추다` [문맥]

정답·적중은 `맞히다`, 비교·조정은 `맞추다`다.

- `정답을 맞추다` → `정답을 맞히다`
- `답안지를 맞춰 보다`는 대조 의미이므로 유지

## G-10. 혼동 동사 1 [문맥]

- 공간 명사 `너머`, 동작 `넘어`: `산 넘어 마을` → `산 너머 마을`
- 교육 `가르치다`, 지시 `가리키다`: `영어를 가리키다` → `영어를 가르치다`

## G-11. 혼동 동사 2 [문맥]

- `붙다`의 사동은 `붙이다`, 그 밖의 보내기·맡기기는 `부치다`.
- 수·양은 `늘리다`, 길이는 `늘이다`.

예: `우표를 부치다` → `우표를 붙이다`, `인원을 늘이다` → `인원을 늘리다`.

## G-12. `낫다/낳다/나다` [강]

- `감기가 낳았다` → `감기가 나았다`
- `아이를 나았다` → `아이를 낳았다`
- `이게 더 낳다` → `이게 더 낫다`

## G-13. `결제/결재`, `지양/지향` [문맥]

- 지불은 `결제`, 승인은 `결재`.
- 피함은 `지양`, 추구는 `지향`.

## G-14. 빈출 오기 [강]

- `됬`→`됐`, `몇일`→`며칠`, `역활`→`역할`, `설레임`→`설렘`
- `오랫만`→`오랜만`, `오랜동안`→`오랫동안`, `희안`→`희한`, `구지`→`굳이`
- `닥달`→`닦달`, `으례`→`으레`, `금새`→`금세`, `폭팔`→`폭발`
- `어의없다`→`어이없다`, 소망 뜻의 `바램`→`바람`, `잠궜다`→`잠갔다`, `담궜다`→`담갔다`
- `왠만`→`웬만`, 단독 `안절부절`→문맥상 `안절부절못하다`

고유명사·브랜드 표기는 이 목록보다 보호 원칙이 우선한다.

## G-15. `-ㄹ게/-ㄹ께`, `-려고/-ㄹ려고` [강]

- `할께` → `할게`, `갈께요` → `갈게요`
- `갈려고` → `가려고`, `먹을려고` → `먹으려고`

## G-16. `왠지/웬` [강]

`왜인지`의 준말 `왠지`를 제외하면 보통 `웬`이다.

- `웬지` → `왠지`
- `왠일`, `왠만하면` → `웬일`, `웬만하면`

## G-17. `어떻게/어떡해` [강]

부사형은 `어떻게`, `어떻게 해`의 준말은 `어떡해`다.

- `이걸 어떡해 하지` → `이걸 어떻게 하지`
- `나 이제 어떻게` → `나 이제 어떡해`

## G-18. `-이/-히` 부사 [문맥]

표준 표기를 사전으로 확인한다.

- `깨끗히`, `곰곰히`, `일일히` → `깨끗이`, `곰곰이`, `일일이`
- `솔직이`, `꼼꼼이` → `솔직히`, `꼼꼼히`

## G-19. 조사와 의존명사 띄어쓰기 [문맥]

조사는 붙이고 의존명사는 띈다.

- `할수 있다`, `먹을것` → `할 수 있다`, `먹을 것`
- `만난지 3년` → `만난 지 3년`
- 조사: `너 뿐`, `학교 에서` → `너뿐`, `학교에서`

같은 형태가 조사·어미·접사일 수 있으므로 문법 기능을 먼저 본다.

## G-20. 단위·호칭과 `안 되다/안되다` [문맥]

- 단위 명사는 원칙적으로 띈다: `한개` → `한 개`, `두살` → `두 살`.
- 성과 이름은 붙이고 호칭은 띈다: `홍 길동`, `홍길동씨` → `홍길동`, `홍길동 씨`.
- 부정·금지는 `안 되다`, 섭섭함이나 일이 잘 풀리지 않음은 `안되다`.

숫자+단위의 관용 표기는 문서 기준과 규범을 함께 본다.

## G-21. 문장 부호 [문맥]

- 직접 인용 종결 부호와 문서의 따옴표·줄임표 스타일을 일관되게 한다.
- 접속사 뒤와 연결어미마다 쉼표를 기계적으로 넣지 않는다.
- 제목·표제어에 마침표를 강제로 붙이지 않는다.
- Markdown 문법, 코드 구두점, URL은 제외한다.

예: `따라서, 결과가 달라졌다` → 쉼이 필요 없으면 `따라서 결과가 달라졌다`.

## G-22. 사이시옷 [문맥]

발음·어원·관용이 얽혀 있으므로 확실한 표기만 고친다.

- `나무가지`, `차집` → `나뭇가지`, `찻집`; 횟수를 뜻하는 `회수` → `횟수`
- `갯수`, `숫놈`, `햇님` → `개수`, `수놈`, `해님`
- `전세방`, `세배돈` → `전셋방`, `세뱃돈`

## G-23. 두음법칙 [문맥]

단어 첫머리, 합성어 내부, 고유명사 여부를 본다.

- `녀자`, `력사`, `류행` → `여자`, `역사`, `유행`
- `년세`, `래일` → `연세`, `내일`
- `남여` → `남녀`; 굳어진 `신여성`은 유지

## G-24. 외래어 표기 [문맥]

표준 표기가 분명한 일상어만 고친다.

- `컨텐츠`, `메세지`, `캡쳐` → `콘텐츠`, `메시지`, `캡처`
- `디지탈`, `화이팅` → `디지털`, `파이팅`

브랜드·서비스·제품의 의도된 표기는 보존한다. 최신 용례가 의심되면 `sources.md`의 1차 자료를 확인한다.

## 문법·비문 검사

- 주어와 서술어, 목적어와 서술어가 명백히 호응하지 않을 때만 최소 수정한다.
- 잘못된 조사, 중복 목적어, 빠진 필수 성분을 문맥 안에서 바로잡는다.
- 명백한 의미 중복은 줄이되 의도적 강조는 보존한다.
- 사동·피동이 문법적으로 가능하다는 이유만으로 취향에 맞춰 바꾸지 않는다.

## 적용 순서

`G-14·G-1·G-7·G-12·G-15·G-17 명백한 표기 → G-2·G-3 → G-4~G-13·G-16·G-18 혼동형 → G-22~G-24 → G-19·G-20 띄어쓰기 → G-21 문장 부호 → 호응 검사` 순으로 본다. 각 수정이 규칙 하나와 연결되지 않으면 롤백한다.
