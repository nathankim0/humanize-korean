# 한국어 번역투 교정 규칙

번역문의 정보 총량과 의미를 보존하면서 원천 언어 간섭과 기계번역체만 다듬는다. 원문이 있어도 재번역하거나 오역을 임의 수정하지 않는다.

## 원천 언어 게이트

- 영한 번역은 무생물 주어, 직역 대명사, 피동, 전치사구, 명사화, 시제 직역을 우선 본다.
- 일한 번역은 피동·사역, 일본어식 조사 결합, 직역 호칭, 문말 표현을 우선 본다.
- 원천 언어가 불명확하면 패턴을 단정하지 않고 한국어 결과의 자연성만 평가한다.
- 한국어의 정상적인 좌분지 수식, 유정물 복수, 실제 매개 의미의 `~을 통해`를 오탐하지 않는다.

## 8대 유형

### 1. 무생물·추상 주어 + 타동사

`이 보고서는 우리에게 보여 준다`, `기술이 변화를 가능하게 한다`처럼 영어 주어 구조가 반복되는지 본다.

- 원문에 실제 사람·기관 주체가 있으면 그 주체를 복원한다.
- 원인 의미면 `~때문에/덕분에/로 인해`, 근거 의미면 `~에 따르면`으로 풀 수 있다.
- 실제 의인화나 학술 개념이면 보존한다.

### 2. 피동·이중 피동

`~되어지다`, `~에 의해`, 불필요한 `~지게 되다`가 반복되면 단일 피동이나 능동으로 바꾼다. 행위자를 모르면 억지로 만들지 않는다.

### 3. 직역 대명사

`그/그녀/그것/그들`이 한국어 담화에서 과도하게 반복되면 생략하거나 명사·호칭으로 복원한다. 생략으로 지시 대상이 모호해지면 유지한다.

### 4. 무생물 복수 `-들`

개별성을 나타내지 않는 추상명사·무생물에 `-들`이 기계적으로 붙었는지 본다. 분포 의미가 있거나 여러 개체를 강조하면 보존한다.

### 5. 외국어식 조사·후치사

`~에 대해`, `~을 통해`, `~에 있어서`, `~와 관련하여`, `~에서의/~으로부터의`를 문맥에 맞는 짧은 조사·절로 바꾼다. 실제 주제·매개·출처 의미는 유지한다.

### 6. 시제·서법 직역

영어의 현재완료·진행형·미래형을 `~해 왔다`, `~하고 있다`, `~할 것이다`로 일대일 대응한 흔적을 본다. 시간 관계와 확실성을 바꾸지 않는 범위에서 한국어 시제로 정리한다.

### 7. 명시화 과잉

원문에 없던 접속어·설명·동어반복 수식이 번역 과정에서 늘었는지 본다. 원천 텍스트가 있으면 대조하고, 없으면 같은 정보를 되풀이하는 구간만 줄인다.

### 8. 명사화·경동사

`회의를 가지다`, `검토를 수행하다`, `영향을 제공하다`, `~의 구현`처럼 동작이 명사 덩어리로 굳었으면 자연스러운 동사로 환원한다. 행위·상태의 차이는 보존한다.

### 보강. `lead to`류 만능 결과 동사

`~로 이어진다`, `~로 이어지게 한다`, `~와 연결된다`가 추상명사 사이의 인과를 반복해서 대신하는지 본다. 실제 사건의 연속·통로·관계라면 정상 용례이므로 보존한다. 번역투라면 원문에 있는 주체와 결과를 직접 서술하거나 `~해서`, `~때문에` 같은 인과절로 풀되, 원문에 없던 인과관계는 만들지 않는다. 동의어 하나로 일괄 치환하지 않는다.

## 교정 순서

`명시화 → 대명사 → 무생물 주어 → 피동 → 조사 → 복수 → 명사화 → 시제 → 긴 관형절` 순으로 처리한다. 앞 단계가 해결한 구간을 다시 쓰지 않는다.

## 원천 텍스트가 있을 때

- 누락·왜곡·오역 의심은 수정하지 말고 `확인 요망`으로 표시한다.
- 원문 대비 늘어난 접속어·수식과 원문 어순 모사를 우선 진단한다.
- 고유명사·전문 용어의 표준 역어를 보호한다.

## 강도

- 보수: 이중 피동, 명백한 직역 대명사 등 강한 신호만 수정한다.
- 기본: 강한 신호와 3회 이상 반복되는 중간 신호를 수정한다.
- 적극: 중간 신호 전반과 필요한 문장 분할까지 허용한다. 정보 보존 기준은 동일하다.

3. 오역을 추측으로 고치거나 정보를 줄이지 않았는가?
4. 강한 번역투가 남지 않았는가?
5. 정상 한국어 표현을 과교정하지 않았는가?
