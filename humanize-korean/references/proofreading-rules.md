# 한국어 교정·교열 규칙

객관적 오류만 고치고 뜻·어투·격식·문단 구조는 유지한다. 규범으로 설명할 수 없는 어색함은 윤문 단계로 넘긴다.

## 분류

| 코드 | 분류 | 예 |
| --- | --- | --- |
| 맞 | 맞춤법 | 되/돼, 안/않, -로서/-로써, -든지/-던지 |
| 띄 | 띄어쓰기 | 의존명사, 조사, 단위, 보조용언 |
| 표 | 표준어 | 바램→바람, 금새→금세, 역할 표기 |
| 외 | 외래어 표기 | 메시지, 콘텐츠, 액세서리, 스케줄 |
| 부 | 문장부호 | 따옴표, 줄임표, 쉼표, 마침표 일관성 |
| 문 | 문법·비문 | 주술·목적어 호응, 조사 오류, 명백한 중복 |

## 핵심 맞춤법

- `되어`로 풀리면 `돼`, 어간이면 `되`: `돼요`, `되면`.
- 부사 `안`과 용언 `않다`를 문맥으로 구분한다.
- 선택·나열은 `-든지`, 과거 회상은 `-던지`.
- 자격·지위는 `-로서`, 도구·수단은 `-로써`.
- 모음이나 `ㄴ` 받침 뒤에는 `율`, 그 밖에는 `률`: 비율, 선율, 확률.
- `왠지`를 제외하면 보통 `웬`: 웬일, 웬만하다.
- `며칠`이 표준어다.
- 사이시옷·두음법칙은 예외가 많다. 확신이 없으면 고치지 말고 확인 대상으로 남긴다.

## 띄어쓰기

- 의존명사는 띈다: `할 수 있다`, `온 지`, `먹을 만큼`.
- 조사는 붙인다: `학교에서`, `너마저`.
- 단위 명사는 원칙적으로 띈다: `한 개`, `20명`처럼 숫자와 단위의 관용도 문서 기준에 맞춘다.
- 보조용언은 원칙 띄어쓰기를 우선하되 허용 표기와 기존 문서 스타일을 존중한다.
- 성과 이름은 붙이고 호칭은 띈다: `홍길동 씨`.
- `안 되다`와 합성어·파생어 `안되다`는 문맥으로 구분한다.

## 표준어·외래어

빈출 오류만 기억에 의존해 바로잡고, 고유명사·브랜드 표기는 보호한다.

- 바램→바람, 설겆이→설거지, 금새→금세, 역활→역할, 일일히→일일이
- 메세지→메시지, 컨텐츠→콘텐츠, 악세서리→액세서리, 스케쥴→스케줄

목록에 없거나 최근 용례가 의심되면 국립국어원 표준국어대사전·우리말샘·외래어 표기 용례 같은 1차 출처를 확인한다. 확인할 수 없으면 수정하지 않는다.

## 문장부호

- 문서의 기존 따옴표·가운뎃점·줄임표 스타일을 일관되게 유지한다.
- 연결어미마다 쉼표를 찍지 않는다. 다만 의미 구분이나 삽입절에 필요한 쉼표는 보존한다.
- 제목·표제어에는 마침표를 강제로 붙이지 않는다.
- Markdown 문법, 코드 구두점, URL은 교정 대상에서 제외한다.

## 문법·비문

- 주어와 서술어, 목적어와 서술어가 명백히 호응하지 않을 때만 최소 수정한다.
- 잘못된 조사, 중복 목적어, 빠진 필수 성분을 문맥 안에서 바로잡는다.
- `역전 앞`, `미리 예습` 같은 명백한 의미 중복은 하나로 줄이되 의도적 강조는 보존한다.
- 사동·피동이 문법적으로 가능하다는 이유만으로 문체 취향에 맞춰 바꾸지 않는다.

## 교정 판단 순서

1. 오류 후보를 `맞/띄/표/외/부/문` 중 하나로 분류한다.
2. 규정상 복수 표기가 허용되는지 본다.
3. 뜻·강조·문체가 바뀌지 않는 최소 수정안을 고른다.
4. 애매하면 원문을 유지하고 `확인 요망`에 근거와 선택지를 적는다.
5. 결과에서 각 교정이 규칙 하나와 연결되는지 검증한다. 연결되지 않으면 롤백한다.
