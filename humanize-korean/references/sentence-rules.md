# 문장 룰북: 군더더기와 직역 구조 덜어 내기

문장 단위의 군더더기와 영어·일본어 직역 구조를 함께 다룬다. 규칙은 `im-ai-copyeditor`의 문장 룰북을 바탕으로 의미 보존 게이트를 강화해 통합했다. 아래 예시는 처방의 방향일 뿐 일괄 치환표가 아니다.

## 판단 원칙

- 먼저 뺄 수 있는 군더더기를 찾되, 문법 기능·강조·양태·개별성·전문 의미가 있으면 보존한다.
- 주어와 서술어를 또렷하게 만들되 원문에 없는 주체·원인·구체 정보를 만들지 않는다.
- **강**은 대체로 수정하고 **중**은 반복·장르·문맥을 보고 수정한다.
- 규칙 이름과 맞는 표현이 보여도 뜻이 흔들리면 `변경없음`으로 둔다.

## S-1. `적·의·것·들` [중]

없어도 뜻과 문법 기능이 유지되는 접미사 `-적`, 관형격 조사 `-의`, 의존명사 `것`, 복수 접미사 `-들`을 덜어 낸다.

- `환경적 요인` → `환경 요인`
- `교사와의 상담이 필요하다` → `교사와 상담할 필요가 있다`
- `운동하는 것이 중요하다` → `운동이 중요하다`
- `여러 학생들이 모였다` → `여러 학생이 모였다`

`법적인 책임`, `그의 책`, `먹을 것`, 개별 구성원을 강조하는 `학생들`처럼 기능이 살아 있으면 유지한다.

## S-2. 군더더기 `있다`와 `-어 있는` [중]

상태·존재 의미가 없는 틀 짓기만 줄인다.

- `벽에 걸려 있는 그림` → `벽에 걸린 그림`
- `길이 막혀 있는 상태다` → `길이 막힌 상태다`
- `발표가 있을 예정이다` → `발표할 예정이다`

실제 존재, 소유, 지속 상태를 나타내면 보존한다.

## S-3. 군더더기 `-ㄹ 수 있다` [중]

가능·능력·허가가 핵심이 아닐 때만 단순화한다.

- `읽을 수 있는 책이 많다` → 문맥상 능력이 아니면 `읽을 책이 많다`
- `성공할 수 있는 가능성` → `성공할 가능성`

불확실성이나 접근 가능성을 나타내면 삭제하지 않는다.

## S-4. `시작하다` 오용 [강]

순간 동사나 이미 변화가 끝난 상태에 `-기 시작하다`를 기계적으로 붙인 표현을 바로잡는다.

- `그 순간 긴장되기 시작했다` → `그 순간 긴장됐다`
- `분위기가 싸늘해지기 시작했다` → 실제 점진 변화가 아니면 `분위기가 싸늘해졌다`

과정의 시작이 정보면 보존한다.

## S-5. `-시키다` 남용 [중]

별도 사동 의미가 없는데 한자어에 붙인 `-시키다`를 `-하다`로 정리한다.

- `그를 설득시켰다` → `그를 설득했다`
- `환경을 개선시켰다` → `환경을 개선했다`

다른 주체에게 실제로 행동하게 한 사동이면 유지한다.

## S-6. 이중 피동과 군더더기 피동 [강]

피동 표지를 한 번만 쓴다.

- `닫혀진 문` → `닫힌 문`
- `쓰여진 글` → `쓰인 글`
- `보여지는 풍경` → `보이는 풍경`
- `되어지다` → 문맥에 맞는 단일 피동 또는 능동

## S-7. 영어식 피동과 `~에 의해` [강]

행위자가 원문에 있을 때만 주어로 복원한다. 행위자를 모르면 만들지 않는다.

- `시민협회에 의해 보호되었다` → `시민협회가 보호했다`
- `감독에 의해 발탁된 선수` → `감독이 발탁한 선수`
- `사고에 의한 정체` → `사고로 생긴 정체`

행위자 은폐가 필요한 법률·학술 문맥이나 피동 자체가 초점이면 보존한다.

## S-8. 무생물·추상 주어 [강]

영어식 무생물 주어에 인지·발화 타동사를 붙인 구조를 한국어답게 푼다.

- `이 그래프는 매출 증가를 보여 준다` → `이 그래프를 보면 매출이 늘었다`
- `연구는 ~라고 말한다` → `연구에 따르면 ~다`
- `데이터가 급증을 나타낸다` → `데이터를 보면 급격히 늘었다`

문학적 의인화나 학술 분야의 정상 용례면 유지한다.

## S-9. 사역 `만들다` 직역 [강]

영어 `make + 형용사`의 흔적이면 자동사·형용사로 환원한다.

- `세상을 더 따뜻하게 만들었습니다` → 원래 인과가 같을 때 `세상이 더 따뜻해졌습니다`
- `그 소식이 나를 행복하게 만들었다` → `그 소식에 나는 행복했다`

누가 의도적으로 무엇을 제작·조성했다는 뜻이면 보존한다.

## S-10. `가지다`와 존재문 직역 [강]

영어 `have/there is`를 직역한 특성·동작성 명사를 자연스러운 술어로 바꾼다.

- `강한 경쟁력을 가지고 있다` → `경쟁력이 강하다`
- `그녀는 두 명의 딸을 가지고 있다` → `그녀에게는 딸이 둘 있다`
- `회의를 가지다` → `회의를 열다`
- `많은 문제가 있다` → 문맥상 `문제가 많다`

실제 소유·보유나 존재가 초점이면 유지한다.

## S-11. 막연한 `대해/대한` [중]

주제 의미가 불필요하면 목적어와 구체 동사를 살린다.

- `그 계획에 대해 설명했다` → `그 계획을 설명했다`
- `그 사건에 관하여 조사 중이다` → `그 사건을 조사하고 있다`

`태도에 대한 연구`처럼 관계를 표시해야 정확하면 보존한다.

## S-12. `-로부터` 분해 [중]

출발점·행위자·비교 기준에 따라 `-에게`, `-와`, `-에서`로 나눈다.

- `선배로부터 조언을 들었다` → `선배에게 조언을 들었다`
- `과거로부터의 교훈` → `과거에서 얻은 교훈`
- `한강으로부터 10킬로미터` → `한강에서 10킬로미터`

물리적·시간적 출발점이 강조되면 보존한다.

## S-13. 전치사 직역 [중]

`~로 인하여`, `~을 통해`, `~을 위해`, `~와 함께`, `~에 있어`, `~의 경우`를 뜻에 맞는 조사·어미·동사로 푼다.

- `폭우로 인하여 행사가 취소되었다` → `폭우로 행사가 취소됐다`
- `데이터 분석을 통해 인사이트를 얻는다` → `데이터를 분석해 인사이트를 얻는다`
- `예약을 하기 위해 전화를 걸었다` → `예약하려고 전화를 걸었다`
- `커피와 함께 책을 읽었다` → 실제 동시 행동이면 `커피를 마시며 책을 읽었다`
- `인생에 있어 가장 중요한 것` → `인생에서 가장 중요한 것`
- `주말의 경우 사람이 많다` → `주말에는 사람이 많다`

실제 매개·목적·동반 의미가 있으면 유지한다.

## S-14. 조사 정밀성 [중]

유정물·무정물, 위치·방향·목적에 맞게 조사를 고친다.

- `정부에게 건의했다` → `정부에 건의했다`
- `도서관을 다닌다` → `도서관에 다닌다`
- `하늘에 연을 날렸다` → 문맥상 `하늘로 연을 날렸다`

기관을 사람 집단으로 의인화하는 정상 용례 등은 문맥을 본다.

## S-15. 관사 직역 [중]

영어 관사에서 온 `그`, `어떤`, `어느`, `하나의`가 지시·수량 기능 없이 붙었으면 뺀다.

- `그 어떤 말도 통하지 않았다` → `어떤 말도 통하지 않았다`
- `그 자신의 잘못` → `자신의 잘못`
- `하나의 좋은 예다` → `좋은 예다`

특정 대상이나 수량 `하나`가 정보면 보존한다.

## S-16. 직역 대명사와 잉여 접속 [중]

한국어 담화에서 주어가 명확할 때 반복 대명사를 생략하거나 명사·호칭으로 복원한다. 목표 삭제 비율은 두지 않는다.

- `존은 피곤했다. 그는 자리에 앉았다` → 지시가 명확하면 `존은 피곤했다. 자리에 앉았다`
- `메리는 그녀의 어머니에게 전화했다` → `메리는 어머니에게 전화했다`
- `책을 읽었고 그리고 잠이 들었다` → `책을 읽다가 잠이 들었다`

생략하면 지시 대상이 모호해지면 유지한다.

## S-17. `-의` 연쇄와 명사화 [중]

영어 `of`나 일본어 `の`를 옮긴 조사 연쇄와 동작을 숨긴 명사 덩어리를 푼다.

- `그녀의 두 명의 딸의 사진` → 문맥상 `그녀의 두 딸 사진`
- `환경 보호에 대한 인식의 제고가 필요하다` → 원래 의무 강도를 유지해 동사 중심으로 풀기

소유·소속 관계가 모호해지거나 정의된 전문 용어면 유지한다.

## S-18. 시제·진행·완료 직역 [중]

과거 표지 중복과 영어 진행·완료의 일대일 대응을 시간 관계가 같은 범위에서 단순화한다.

- `지난주에 읽었던 책` → 완료된 단일 사건이면 `지난주에 읽은 책`
- `전철이 도착하는 중입니다` → 실제 진행 구간이 아니면 `전철이 도착합니다`
- `그동안 많은 노력을 기울여 왔다` → 지속성이 핵심이 아니면 `그동안 많이 노력했다`

지속·경험·완료가 정보면 보존한다.

## S-19. 안긴 의문절 [중]

내포된 의문절은 `-는가/-인가`보다 `-는지/-ㄹ지`를 쓴다.

- `왜 실패했는가를 분석했다` → `왜 실패했는지 분석했다`
- `그것이 옳은가를 따졌다` → `그것이 옳은지 따졌다`

제목·직접 의문이나 학술 문체의 허용 범위는 문맥을 본다.

## S-20. 영문법 구문 직역 [중]

가주어, `one of`, 비교급 중복, `so~that`, 관계절 좌향 수식을 한국어 어순과 어미로 푼다.

- `그가 거짓말했다는 것은 분명하다` → `그는 분명히 거짓말했다`
- `가장 뛰어난 선수들 중 하나다` → 뜻이 같으면 `손꼽히는 선수다`
- `나보다 더 키가 크다` → `나보다 키가 크다`
- `너무 피곤해서 일어날 수 없었다` → `너무 피곤해서 못 일어났다`
- `감탄하지 않을 수 없었다` → 원래 강도가 같으면 `절로 감탄했다`

논리적 범위·비교 집합·불가능성의 강도가 달라지면 원문을 유지한다.

## S-21. 괄호 줄이기 [중]

설명 괄호가 반복되고 본문 흐름을 끊을 때 문장 안에 자연스럽게 녹인다.

- `사용자(특히 초보자)는 헷갈린다` → `초보 사용자는 헷갈린다`
- `예외도 있다(드물지만)` → `드물지만 예외도 있다`

약어 정의, 인용 출처, 법적 표기, 수학·코드, 정확한 보충 정보는 보호한다.

## S-22. 쉼표 줄이기 [중]

연결어미마다 찍은 쉼표나 문장 뼈대를 끊는 쉼표를 줄인다.

- `그는 빠르게, 그리고 정확하게, 일을 끝냈다` → `그는 빠르고 정확하게 일을 끝냈다`
- `자료를 모으고, 정리하고, 분석해서, 결론을 냈다` → `자료를 모아 정리하고 분석해 결론을 냈다`

나열, 삽입, 호흡, 의미 구분에 필요한 쉼표 수를 목표치에 맞춰 삭제하지 않는다.

## S-23. 만연체 추리기 [중]

연결어미와 곁가지가 겹쳐 핵심이 흐려진 문장을 뜻의 손실 없이 정리한다. 일반 윤문에서는 필요한 경우 합치거나 나눌 수 있지만, 장문 작업표 파이프라인에서는 한 세그먼트 안에서만 고친다.

- `회의가 길어졌고, 사람들이 지쳤으며, 결론도 안 났고, 시간만 흘렀다` → `회의가 길어져 다들 지쳤고 결론은 못 냈다`

각 절이 독립 정보면 삭제하지 않는다.

## S-24. 에두르는 형식 [중]

`~축에 들다`, `~라고 할 수 있다`, `~하는 편이다`가 불확실성이 아닌 군더더기일 때 직접 쓴다.

- `효과가 있다고 할 수 있다` → 근거와 확실성이 같으면 `효과가 있다`

실제 상대 평가나 불확실성을 나타내면 보존한다.

## S-25. 풀어 쓴 표현을 적확한 말로 [중]

뜻이 완전히 같은 범위에서 긴 표현을 정확한 낱말로 줄인다.

- `영어를 옮긴 듯한 말투` → `영어 번역체`
- `매우 빠른 속도로 처리한다` → `빠르게 처리한다`
- `글을 읽는 사람의 입장에서` → `독자 입장에서`

어휘를 화려하게 보이게 하려고 낯선 동의어로 치환하지 않는다.

## S-26. 일본어투·행정어 [중]

굳은 일본어식 서술 틀과 어려운 행정어를 장르에 맞는 쉬운 말로 푼다.

- `규정에 의거하여 처리한다` → `규정에 따라 처리한다`
- `검토를 요한다` → `검토해야 한다`
- `금번 회의` → `이번 회의`
- `상기 내용을 참고 바랍니다` → `위 내용을 참고해 주세요`
- `~에 위치하다` → 문맥상 `~에 있다`
- `~에 다름 아니다` → `바로 ~다`

법령·계약·정의된 행정 용어는 자동으로 바꾸지 않는다.

## 적용 순서

`S-1~S-6 군더더기·피동 → S-7~S-20 직역 구조 → S-24~S-26 에두름·어휘·행정어 → S-21~S-23 괄호·쉼표·만연체` 순서로 본다. 앞 단계에서 자연스러워졌으면 뒤 단계는 건너뛴다. 마지막에 모든 변경이 원문 명제와 같은지 대조한다.
